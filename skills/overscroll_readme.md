# Overscroll Effect Skill
A premium "rubber-band" overscroll effect for web applications, inspired by iOS and EverywhereMe.

## Features
- Smooth pull-to-refresh style physics.
- High-fidelity bounce back using GSAP Elastic ease.
- Supports both mouse wheel and touch interactions.
- Prevents scroll leakage to parent containers.

## Usage
Include the script and initialize with a selector:
```javascript
OverscrollEffect.init('.my-scroll-container');
```

## Dependencies
- GSAP 3.x
